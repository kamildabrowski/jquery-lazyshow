INSTALL

Use plugin before </body> as last element - is important to work properly.

-------
HOW TO USE?

Two class 
  Parent: lazyshow tag cant be display:none;
  Child: lazyshow-box is auto hide after loaded script.

-------
EXAMPLE:

<div class="lazyshow">
	<div class="lazyshow-box">
	
	</div>
</div>

-------
ATTENTION:

Important is sequence in code html is must be the same as rendered website.
